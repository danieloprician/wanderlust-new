## Image Optimization
- Use WebP format for best compression
- Fallback to AVIF when supported by Next.js
- Keep file sizes under 200KB
- Use descriptive, SEO-friendly filenames
- Add descriptive alt text in code

## Free Stock Images
You can find free cabin/nature images on:
- Unsplash (https://unsplash.com/s/photos/cabin)
- Pexels (https://www.pexels.com/search/mountain-cabin/)
- Pixabay (https://pixabay.com/images/search/log-cabin/)

Remember: Use only images you have rights to!
